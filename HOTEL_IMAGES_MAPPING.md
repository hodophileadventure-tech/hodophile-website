# Hotel Images URL Mapping

This document contains image URLs for all hotels and rooms in the system. You can update the `hotels.ts` file with these URLs.

## Format
Add `image: "URL"` to each hotel and room object.

## USAGE

Each hotel entry should look like:
```typescript
{
  id: "hotel-id",
  name: "Hotel Name",
  city: "City",
  image: "HOTEL_IMAGE_URL",  // ← Add here
  rooms: [
    {
      name: "Room Type",
      price: 10000,
      image: "ROOM_IMAGE_URL",  // ← Add here
    }
  ]
}
```
