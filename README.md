# Hodophile Tours and Travels

SEO-first Next.js website for domestic travel in Pakistan. The site is built with the App Router, TypeScript, Tailwind CSS, static routes, sitemap, robots, canonical metadata, and JSON-LD structured data.

## Pages

- `/`
- `/about`
- `/tours`
- `/destinations`
- `/blog`
- `/contact`
- `/inquiry`

## Development

```bash
npm run dev
```

## Production build

```bash
npm run build
```

## Hostinger deployment

This project is ready for Hostinger VPS deployment. Set `NEXT_PUBLIC_SITE_URL` to your real domain before deploying so canonical URLs, sitemap entries, and structured data point at the correct site address.

## SEO notes

- All main pages export metadata.
- `sitemap.xml` and `robots.txt` are generated by the App Router.
- JSON-LD is included in the root layout for the travel agency and website schema.
- The color system uses solid black and `#FCC000`.
